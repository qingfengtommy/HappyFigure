## Diagram Workflow

### EXPLORE / DESIGN

- Spawn `@method-explore` to read the proposal and write:
  - `method_exploration_report.md` — data file inventory and architecture context
  - `method_description.md` — detailed component/connection descriptions for SVG generation
- If `results_dir` is provided, also optionally spawn `@data-explore` for visualization context.
- Read the method description yourself, then write:
  - `diagram_design_spec.md`
  - `design_summary.json`
